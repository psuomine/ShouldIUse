## Getting Started

### Prerequisites

To develop this project, you have to add .env file root of the project.

1. get a GitHub access token from your GitHub account.

```
REACT_APP_GITHUB_ACCESS_TOKEN=*ACCESS TOKEN HERE*
NODE_PATH="src/"
```

Run project

```
yarn start
```

### Run tests

Run tests with a next command

```
yarn test
```
